## 📊 Using the Application

You can start the tool by double-clicking the shortcut on your desktop. The main interface opens and displays a dashboard populated with flight data. 

### Navigating the Dashboard

The dashboard provides several views to help you understand flight performance:

*   Airline Performance: Select an airline from the dropdown menu to see their specific on-time records.
*   Airport Insights: View delays by specific airport locations.
*   Seasonal Trends: Track performance spikes during different months or holidays.
*   Delay Root Causes: Identify common reasons for departures or arrivals falling behind schedule.

### Importing New Data

The software allows you to import external files as long as they follow the standard format. Click on the File menu in the top left corner, then select Import. Choose your CSV file to merge that data into your current view.

## 🔍 Understanding the Data

The application organizes data into clear visual formats. You see bar charts, line graphs, and map overlays. 

*   The Bar Charts compare delay frequencies across different airlines.
*   The Line Graphs highlight changes in performance over a twelve-month period.
*   The Map Overlays indicate which regions experience the most significant flight interruptions.
